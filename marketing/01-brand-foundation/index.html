Voice

01.04
Do

Degen-confident. Data-grounded.

  • Lead with the data: "$2.4M moved on Trump-Powell in 18 minutes."
  • Use the Money / Mouth / Edge vocabulary across every surface.
  • Short, declarative sentences. Anton headlines, hairline rules, restraint.
  • Mono uppercase for labels (VOLUME 24H, DIVERGENCE).
  • Concede uncertainty when honest — credibility compounds.
  • Address the reader as someone who already gets it. No explainers in the hero.
Don't

Generic crypto slop.

  • No "TO THE MOON," no rockets, no laser eyes, no 🚀💎.
  • No purple-on-white SaaS gradients. No "AI-powered" boilerplate.
  • No Inter / Roboto / system fonts as the display face.
  • No "DYOR" "WAGMI" "GM" decoration — vocabulary, not jewelry.
  • Never write $1,234.56 in a data row — use compact ($1.2K).
  • Never use lime on lime, magenta on lime, or any color combo that isn't in the palette.
