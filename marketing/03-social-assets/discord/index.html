Welcome + walkthrough + roles + conduct

**Welcome to $EDGE TERMINAL.**

This is the control room for traders who watch prediction markets the same way fund desks watch ticker tape.

**Money. Mouth. Edge.**
— Money: every Polymarket + Kalshi market, whales surfaced in real time.
— Mouth: aggregated sentiment from X, Farcaster, Reddit.
— Edge: the live gap between the two. That's where you bet.

Site: https://www.thepolyedge.com/
Token: $EDGE on Solana (pump.fun launch).

────────────────────────────────

**CHANNEL WALKTHROUGH**

# intel-room          → forum threads. One market per thread. Bring receipts.
# whales-alerts       → bot-posted. $25K+ wallet moves on Poly / Kalshi.
# divergence          → bot-posted. Money/Mouth gaps ≥0.15.
# polymarket          → discussion of specific Poly markets.
# kalshi              → discussion of specific Kalshi markets.
# picks               → call your shot. Pinned leaderboard. Receipts mandatory.
# track-record        → public picker algo log. No editing, no deleting.
# trade-tape          → cross-venue trade tape. Watch only.
# dev-log             → what we shipped this week. PRs welcome.
# lounge              → off-topic.
# holders-only        → token-gated. AMA, alpha, brand-account engagement.
# governance          → holders-only. Vote on which markets get bot coverage.

────────────────────────────────

**ROLES** (claim in #role-pickup)

@Watcher          → default on join. Read all public rooms.
@Builder          → contributing to data, code, content.
@Holder           → ≥1 $EDGE. Unlocks #holders-only.
@Holder-1K        → ≥1,000 $EDGE. Webhook access to whale alerts.
@Holder-10K       → ≥10,000 $EDGE. Push notifications + governance vote.
@Trader-100K      → ≥100,000 $EDGE. Custom firehose thresholds + watchlists + deep Rug scan.
@Whale            → ≥1M $EDGE. Direct line. Help shape the roadmap.
@Mod              → community ops.


────────────────────────────────

**CODE OF CONDUCT**

1. Bring receipts. Screenshots, links, on-chain hashes, market IDs — not vibes.
2. Disagree with the data, not the person.
3. No shilling unrelated tokens. We're a prediction-market room, not a chain-shill room.
4. No DM scams. We will never DM you about claiming, airdrops, or "verifying your wallet."
5. The picker algorithm log is public and immutable. Don't ask us to edit it after a bad call.
6. English is the working language.
7. Mark decided markets RESOLVED with the outcome.


We're early. Build with us.
