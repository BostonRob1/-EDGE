# 60-SECOND LAUNCH TRAILER — "MONEY / MOUTH / EDGE"

**Format:** 16:9 master · 1:1 + 9:16 reframes
**Total runtime:** 60.0s
**VO talent:** an Operator (or ElevenLabs clone — see production-notes.md)
**Music:** Ambient industrial · 95–105 BPM · low sub · sparse percussion · NO drops
**Color:** Pure Void (#050507) backgrounds throughout. Lime #C4FF00 as the only chromatic accent except for the divergence moment (Shot 06 — Magenta #FF006E).
**SFX:** Terminal keystrokes, single bell-tick on data prints, sub-bass whump on each scene cut.

---

## SHOT 01 — 0.0s–4.0s · COLD OPEN

**VISUAL:** Pure black. A single white cursor blinks dead-center. At 1.0s, type sound. The word `MONEY` types itself in JetBrains Mono 14px, lime. Cursor advances. At 2.5s, `MOUTH` types below it in white. At 3.5s, `EDGE` types in magenta — but only the word, no value yet. Hold.

**AUDIO / VO:** (Subtle room tone + 3 mechanical key clicks, one per word.) No VO.

**CAPTION:** None.

**NOTES:** This is the only shot with no music. Silence + keystrokes. Sets the terminal tone.

## SHOT 03 — 9.0s–14.0s · MOUTH

**VISUAL:** Split-screen. Left half: the lime Money chart from Shot 02, simplified. Right half: tweet wall from Shot 03, blurred. A magenta vertical line slides in down the center seam. Labels animate in: left `MONEY +18.4%`, right `MOUTH -32% SENTIMENT`. Below the line in mono: `DIVERGENCE 0.72`.
