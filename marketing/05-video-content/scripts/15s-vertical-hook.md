## PLATFORM NOTES

- **TikTok:** Add 100px bottom safe area (avoid caption overlap with the TikTok UI).
- **IG Reels:** Add 250px bottom safe area for the IG action bar.
- **X vertical:** No safe-area changes needed; X overlays minimal UI.
- **Captions:** Use TikTok's native caption tool ON TOP of the burned-in captions — TikTok's algorithm rewards native captions.
- **First frame:** TikTok uses your first frame as the thumb. The magenta line + caption should be visible at frame 0.
